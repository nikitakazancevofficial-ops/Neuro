# Changelog

## v1.1.0 - Local AI Music

- Added the AI Music studio for local song generation with ACE-Step 1.5.
- Added chat-to-music routing: Neuro can plan lyrics, style, duration and metadata before synthesis.
- Added manual music mode for user-written style prompts and structured lyrics.
- Added optional song duration input. When empty, Neuro chooses a natural duration up to the ACE-Step 10-minute runtime limit.
- Added music library, playback screen, synchronized lyric timeline and regenerate button.
- Added semantically related cover artwork generation with fresh visual variants.
- Added one-file Windows launcher flow through `start_neuro.bat`.
- Kept the main `Neuro Server` console visible so users can see the phone URL and backend logs.
- Added Android APK release workflow documentation and refreshed GitHub screenshots.

## v1.0.0 - Initial Portfolio Release

- Local Android chat interface with LM Studio backend.
- Saved memory, personalization and runtime language switching.
- Local image understanding and FLUX image generation workflow.
- Faster Whisper voice transcription.
