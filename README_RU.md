Neuro - активно развивающийся offline-first AI-помощник и портфолио-проект. Приложение объединяет нативный Android-интерфейс, локальные языковые модели, постоянную память, анализ изображений, локальную генерацию картинок, распознавание голоса и необязательный backend ACE-Step 1.5 для музыки. Сейчас это ранняя стадия проекта, но основа уже работает и быстро развивается.

## Возможности

- Локальное выполнение: личные чаты и созданные изображения остаются на вашем компьютере
- Нативный Android-клиент на Kotlin и Jetpack Compose
- Потоковые ответы по SSE
- Выбор локальных моделей через LM Studio
- Сохраненная память, персонализация и контекст недавних чатов
- Анализ изображений мультимодальными моделями
- Локальная генерация изображений FLUX.2 Klein с референсами и повторной визуальной проверкой
- Полноэкранный просмотр изображений с масштабированием, ограниченным перемещением, скачиванием и отправкой
- Распознавание голоса Faster Whisper
- Локальная генерация песен ACE-Step 1.5, аудиокаверы, новые варианты обложек и синхронизированный текст
- Выбор языка интерфейса и ответов Neuro прямо в приложении

## Языки

Полноценный перевод интерфейса доступен для:

- Русского
- Английского
- Украинского

Neuro также умеет отвечать на немецком, испанском, французском, итальянском, португальском, польском, турецком, китайском и японском языках. Для этих языков интерфейс пока использует английский перевод.

## Планы развития

Neuro активно развивается. В планах:

- Больше музыкальных инструментов: stems, более точные каверы и редактирование аранжировки
- Более мощный AI Agent для многошаговых задач
- Дальнейшая полировка Android-интерфейса
- Более простой запуск и управление моделями
- Расширение локальных мультимодальных возможностей

## Архитектура

```text
Android-приложение (Jetpack Compose)
        |
        | HTTP + SSE
        v
FastAPI backend
   |           +--> FLUX.2 Klein image worker
   |           +--> ACE-Step 1.5 music worker
   |
   +--> OpenAI-совместимый API LM Studio
   +--> Модель Faster Whisper
```

Большие модели, личная память, загруженные и созданные изображения, а также локальные Python-окружения намеренно не попадают в Git.

## Требования

- Windows 10 или новее
- Android Studio с Android SDK 34
- JDK 17
- Python 3.10
- Python 3.12 для необязательной локальной генерации музыки ACE-Step 1.5
- LM Studio с включенным локальным OpenAI-совместимым сервером
- NVIDIA GPU рекомендуется для локальной генерации изображений FLUX

## Быстрый старт

### Самый простой путь для пользователей

1. Скачайте свежий APK в [GitHub Releases](https://github.com/nikitakazancevofficial-ops/Neuro/releases/latest) и установите его на Android-телефон.
2. Скачайте или клонируйте этот репозиторий на Windows-ПК, где будет работать локальный AI backend.
3. Запустите LM Studio, включите OpenAI-совместимый локальный сервер `http://127.0.0.1:1234/v1` и загрузите одну из рекомендованных локальных моделей из выбора моделей в приложении.
4. Дважды нажмите `start_neuro.bat`.
5. Скопируйте адрес из консоли, например `http://192.168.1.10:3510`, в `Настройки -> Подключение к ПК` в Android-приложении.

`start_neuro.bat` - основной запуск одним файлом. Он оставляет главное окно
`Neuro Server` видимым, показывает адрес для телефона, запускает FLUX и ACE-Step,
если они установлены, и пишет логи в `server\neuro-server-current.*.log`.

### 1. Настройте Android-клиент для разработки

Создайте локальную Android-конфигурацию:

```powershell
Copy-Item local.properties.example local.properties
```

Укажите `sdk.dir` и LAN-адрес компьютера в `local.properties`:

```properties
neuro.serverUrl=http://192.168.1.10:3510
```

### 2. Настройте LM Studio и backend

Создайте локальную конфигурацию backend:

```powershell
Copy-Item run_server.local.bat.example run_server.local.bat
```

Укажите тот же LAN-адрес в `PUBLIC_SERVER_URL`. В `LLM_BASE_URL` укажите адрес LM Studio.

### 3. Запустите Neuro

```powershell
.\start_neuro.bat
```

Скрипт создаст `server\.venv`, установит зависимости и запустит API на порту `3510`. Если FLUX.2 Klein уже установлен, локальная генерация изображений включится автоматически. В консоли появятся адреса в блоке `Введите в приложении`.
После запуска останется небольшое окно `NEURO IS READY` с актуальным адресом
для телефона. Главное окно `Neuro Server` остаётся видимым, чтобы пользователь
видел состояние backend и логи. Если закрыть только информационное окно, сервисы
продолжат работать.

На экране входа Android-приложения нажмите `Настроить подключение к ПК`, вставьте адрес из консоли и нажмите `Проверить подключение`. Позже адрес можно изменить в настройках Neuro.

### 4. Необязательно: установите локальную генерацию изображений

```powershell
.\setup_flux_klein.bat
.\start_neuro.bat
```

### 5. Необязательно: подготовьте локальную генерацию музыки

```powershell
.\setup_acestep_music.bat
.\start_neuro.bat
```

Скрипт клонирует официальный исходный код ACE-Step 1.5, создаст отдельное
окружение для музыки и скачает официальный checkpoint. Backend уже предоставляет
API для генерации песен, каверов, прогресса, воспроизведения и временной шкалы
текста. Модели и все тяжёлые кэши музыки останутся внутри
`E:\Portfolio\server`. Установщик связывает официальный каталог checkpoints
ACE-Step с каталогом моделей Neuro, поэтому первый запрос не скачивает вторую
копию файлов.

`start_neuro.bat` также проверяет системный диск. Если Windows недавно записала
ошибки ввода-вывода, проблемы файловой системы или критически малый остаток
свободного места, Neuro запускает лёгкий backend чата без FLUX и ACE-Step до
проверки системного SSD.

`start_neuro.bat` и `start_neuro_all.bat` запускают backend чата, FLUX и ACE-Step вместе, когда проверка
системного диска проходит успешно. Пока Windows записывает связанные с диском
критические сбои, программного обхода защиты намеренно нет. Подробнее:
Тяжёлые воркеры автоматически выгружаются после каждого результата и повторно
запускаются по запросу. Если LM Studio зависает при подготовке песни, через 90
секунд Neuro продолжает работу с локальным музыкальным планом вместо вечного ожидания.
В AI Music также есть прямой ручной режим для собственного стиля и готового
текста с разметкой куплетов и припевов. Neuro сам выбирает естественную
длительность песни, если пользователь не указал её вручную. Официальный runtime
ACE-Step поддерживает песни длительностью до 10 минут. После синтеза локальный
CPU Whisper выравнивает текст по сгенерированному вокалу для режима караоке.
Готовый трек можно перегенерировать с тем же стилем и текстом, но с новой
музыкальной вариацией и новой смыслово похожей обложкой.
[документация музыкального backend](docs/MUSIC_BACKEND.md).

### 6. Соберите Android-приложение

```powershell
.\gradlew.bat :app:assembleDebug
```

APK появится в:

```text
app\build\outputs\apk\debug\app-debug.apk
```
